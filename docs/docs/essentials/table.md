## Alternating rows

You can use the `wa-alternating-row-colors` class to add alternating row colors to your table:

```html {.example}
<table class="wa-alternating-row-colors">
  <caption>I'm just a table</caption>
  <thead>
    <tr>
      <th>Column 1</th>
      <th>Column 2</th>
      <th>Column 3</th>
      <th>Column 4</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Cell</td>
      <td>Cell</td>
```
